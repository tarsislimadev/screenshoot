## how to use

### list releases

```
bash env/releases/list.sh
```

### download release files

```
bash env/releases/download.sh  v2023-11-02-13-03 
```

### download latest release files

```sh
bash ./env/releases/download.sh $( bash ./env/releases/list.sh ) downloads
```
